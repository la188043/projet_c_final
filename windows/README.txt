Si le main.exe s'exécute et se ferme presque instantanément ou n'affiche rien après quelques secondes :
******************************************************************************************************

1. Clic droit sur "main.exe" --> propriétés.
2. Dans l'onglet "compatiblité" cochez la la case "Excécutez ce programme en mode de compatiblité pour : ".
3. Ensuite choisissez "Windows 8".
4. Appliquer et ok. Le programme devrait maintenant être fonctionnel.
